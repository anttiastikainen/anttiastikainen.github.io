Customer database project

This is a C# program that utilizes Windows Presentation Foundation (WPF) to build a database management application. The application allows the user to load and display data from a MariaDB database and provides options to remove customers and orders from the database. The user can also add new customers and orders using the provided forms. I made this project to learn using C# and at the same time I learned about how to use database connections and to create user interfaces on C#
